Quick Start

Get started with the CMS:

  1. Access the Admin Panel: /admin (Default credentials: admin/admin123)
  2. Create Content Types: Define your custom content structures
  3. Add Content: Start creating pages, posts, or any custom content
  4. Use the API: Access content via /api/content

API Endpoints

Method Endpoint Description
GET /api/content Get all content with pagination and filtering
GET /api/content/<id> Get specific content by ID
GET /api/content/by-slug/<slug> Get content by slug
POST /api/content Create new content (requires auth)
PUT /api/content/<id> Update content (requires auth)
DELETE /api/content/<id> Delete content (requires auth)
GET /api/search?q=query Search across all content
